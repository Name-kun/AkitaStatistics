# AkitaStatistics
## これはなに
我らが愛し秋田県についての統計情報や天気を表示するだけの~~ゴミ~~神プラグインです。<br>
~~需要は知りません。~~需要しかありません。

## どうやって使うねーん
**/akita help**: ヘルプを表示します。<br>
**/akita info [city]**: 指定した市町村の統計情報を表示します(未指定の場合は秋田県の統計情報)<br>
**/akita weather [city]**: 指定した市町村の今日・明日の天気を表示します(未指定の場合は秋田市の天気)
